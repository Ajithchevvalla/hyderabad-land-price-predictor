# Hyderabad Land Price Prediction Web App

A simple machine-learning inspired web application to estimate land prices in Hyderabad.

## Inputs
- Location
- Land area (sqft)
- Distance to main road
- Road width
- Distance to IT hub
- Nearby facilities score

## Tech Stack
- HTML
- CSS
- JavaScript
- GitHub Pages

## Deployment
1. Upload files to GitHub
2. Go to Settings → Pages
3. Select main branch → root
4. Access live link:
https://YOUR_USERNAME.github.io/REPO_NAME/
